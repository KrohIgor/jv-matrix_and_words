Матрица и Слово
Ваша программа должна получить два параметра из командной строки:

строка размером N ^ 2, описывающая квадратную матрицу символов N * N;

Строка, которая описывает данное слово.

Первая строка преобразуется в матрицу по следующему правилу. Строка «QWEASDZXC» формирует матрицу:

[ 'Q' 'W', 'Е',

'А', 'S', 'D',

'Z', 'X', 'C']

Ваша программа должна вывести на консоль последовательность ячеек, составляющих данное слово.

Каждый следующий символ слова может быть размещен только в соседней ячейке: сверху, снизу, слева или справа от ячейки с предыдущим символом.

Например, если задана матрица «QLGNAEKIRLRNGEAE» и слово «KING», то последовательность ячеек будет [1,2] -> [1,3] -> [0,3] -> [0,2]

После решения задачи тщательно ее протестируйте на разных примерах.
